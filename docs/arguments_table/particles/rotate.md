# 旋转 Rotate

## 简介
使粒子画旋转至任意角度，而不是简单地处于某个平面

*注：当 X、Y、Z 的任意一个位置获得任意值（包括 0）后即启用旋转，即会导致“平面”参数失效*。

## 参数细节
类型：`List<double>`

取值范围：$R$

## 详解

该参数填入的是粒子画旋转后所在平面的[法向量](https://baike.baidu.com/item/%E6%B3%95%E5%90%91%E9%87%8F/1161324)。当然，法向量的正负性也会影响粒子画的翻转。

例如，您想让粒子画斜着面向（朝着）XYZ三个方向的正方向，您可以填：1, 1, 1。法向量的长度并不影响旋转角度，如：2, 2, 2。但 -1, -1, -1 会让粒子翻转。
